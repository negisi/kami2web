# 上高田二丁目町会 Webサイト (kami2web)

このリポジトリは、「上高田二丁目町会」の公式Webサイトのソースコードを管理するプロジェクトです。
本プロジェクトは、高速なフロントエンドビルドツールである [Vite](https://ja.vitejs.net/) を使用して構築されています。

## 📂 プロジェクト構成

主なファイルと役割は以下の通りです：
- `index.html`: WebサイトのメインとなるHTMLファイル（お知らせやコンテンツの修正・追加はこちらで行います）
- `style.css`: サイトのデザインやレイアウトを定義するCSSファイル
- `main.js`: サイトの動き（アニメーションなど）を制御するJavaScriptファイル
- `public/`: サイトで使用する画像やアイコンなどの静的ファイルを配置するディレクトリ
- `町会情報まとめ.md`: サイトに掲載する情報（行事予定や概要など）の原稿データ
- `netlify_migration_guide.md`: Netlify移行時の初期設定手順の記録（現在は設定済み）

## 🛠 開発環境のセットアップ（運用担当者の方向け）

このプロジェクトをご自身のPCで動かしたり、修正内容をプレビューしたりするための手順です。
前提として、PCに [Node.js](https://nodejs.org/ja/) がインストールされている必要があります。

### 1. パッケージのインストール
プロジェクトのディレクトリ（この `README.md` があるフォルダ）でターミナル（Macの場合はターミナルアプリ、Windowsの場合はコマンドプロンプトやPowerShellなど）を開き、以下のコマンドを実行します。

```bash
npm install
```
これにより、Viteなど動作に必要なパッケージがインストールされます。（初回のみ実行）

### 2. ローカル開発サーバーの起動
以下のコマンドを実行すると、ご自身のPC上でWebサイトのプレビューを立ち上げることができます。

```bash
npm run dev
```
実行後、ターミナルに表示されるURL（例: `http://localhost:5173/` ）をブラウザで開いてください。
この状態で `index.html` や `style.css` を編集して保存すると、ブラウザの表示が自動的に最新状態に更新され、すぐに変更を確認できます。

## 🚀 ビルドと公開（デプロイ）

本番環境（インターネット上）向けにファイルを最適化する手順です。

### 本番用ビルドの作成
```bash
npm run build
```
このコマンドを実行すると、公開用に最適化されたファイル群が `dist/` フォルダに出力されます。

### ビルド結果のローカル確認
`dist/` に出力された内容が本番環境と同じように正しく表示されるか確認したい場合は、以下のコマンドを実行します。
```bash
npm run preview
```

### 🔄 自動公開（自動デプロイ）について
このサイトは **Netlify** を使用して公開されています。
連携設定は既に完了しているため、**GitHubのリポジトリに変更をプッシュ（アップロード）するだけで、自動的に本番環境へデプロイ（公開）**されます。
毎回手動でビルド（`npm run build`）を行ってアップロードする必要はありません。

## 📝 日常的なコンテンツの更新方法

お知らせの追加や、テキスト・画像の差し替えなどの日常的な更新は、主に以下の手順で行います。

1. **修正する**: `index.html` をエディタで開き、該当するテキストや画像パスを修正します。
2. **確認する**: ターミナルで `npm run dev` を実行し、ブラウザで正しく表示・動作しているか確認します。
3. **反映する**: 変更内容をGitでコミットし、リモートリポジトリ（GitHub等）へプッシュします。
4. **自動デプロイ**: GitHubへプッシュした時点で自動的にビルドが走り、インターネット上の本番環境に反映（デプロイ）されます。
